# Reproducibility

This repository is designed to be reproducible at the workflow level.

A builder should be able to:

1. read the monitoring use case
2. inspect the prompt templates
3. inspect the watch rule schema
4. inspect the mock watch rule and mock alert result
5. follow the happy path demo
6. understand how an alert is triggered from a natural-language request

Minimum reproducibility assets included:

- README
- prompt templates
- watch rule schema
- mock-data examples
- demo flow
- simple source code entry
