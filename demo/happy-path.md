# Happy Path Demo

## User request
Watch this BTC whale wallet and alert me if it transfers out more than 500 BTC within 24 hours.

## Compiled watch rule
The request is compiled by Claw into a structured watch rule.

## Monitoring
The agent checks the target wallet activity.

## Trigger
An outflow event above the configured threshold is detected.

## Result
The system triggers an alert and records an event log.
